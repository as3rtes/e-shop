# e-shop
1. Скопируйте код из файл Main.java
2. Сохраните его в файл с именем Main.java
3. Откройте папку, где лежит ваш файл Main.java
4. Нажмите на адресную строку вверху
5. Напечатайте cmd и нажмите Enter
6. Терминал откроется уже в нужной папке
7. Выполните команды: javac
8. Main.java, java Main
